# Tekzura — Main Website

Premium digital agency marketing site for **Tekzura**. Built with React, TypeScript, and Vite.

Design reference: [Figma — Premium Digital Agency Website](https://www.figma.com/design/3GxfAXC5YzWZwh46HTXzn1/Premium-Digital-Agency-Website)

## Quick Start

```bash
npm install
npm run dev
```

Dev server runs at [http://localhost:3000](http://localhost:3000).

```bash
npm run build    # Output → build/
npm run preview  # Preview production build
```

**Requirements:** Node.js ≥ 22, npm ≥ 8

## Documentation

Full documentation — architecture, pages, components, content, deployment, and maintenance — is in **[DOCUMENTATION.md](./DOCUMENTATION.md)**.

## Site Overview

| Page | Path |
|------|------|
| Home | `/` |
| About | `/about` |
| Services | `/services` |
| Service detail | `/services/:slug` |
| Work / Portfolio | `/work` |
| Insights (Blog) | `/blog` |
| Contact | `/contact` |

## Tech Stack

React 18 · TypeScript · Vite 7 · React Router 7 · Lucide Icons · Plain CSS

Deployed on **Vercel** with SPA rewrites.
